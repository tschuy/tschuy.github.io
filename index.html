Soil Finder

Want to learn about the soils beneath your feet? Soil Finder uses NRCS data, hosted by UC Davis to show you the soils below almost any point in the United States.

Soil Finder uses the Google Maps API for Android to allow a user to tap on a map to choose the location, and the Google Geocoding API to allow a user to enter an address or place name.

You can try out Soil Finder on any Android device running Android 4.0 and up by downloading the latest build from here.
