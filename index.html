Projects
GIMP Plugin Installer

GIMP, the GNU Image Manipulation Program, is a fantastic piece of software. One of its coolest features is its third-party plugins API, which allows users of the software like you and me to take it and make it do what we want it to.

GPI is designed to make managing GIMP plugins as easy as managing Python packages. With a single command, users can install packages from the web, without having to copy files from place to place, and with no hassle.

GPI is live today. You can check it out at gpi.tschuy.com.

Soil Finder

Want to learn about the soils beneath your feet? Soil Finder uses NRCS data, hosted by UC Davis to show you the soils below almost any point in the United States.

Soil Finder uses the Google Maps API for Android to allow a user to tap on a map to choose the location, and the Google Geocoding API to allow a user to enter an address or place name.

You can try out Soil Finder on any Android device running Android 4.0 and up by downloading the latest build from here.

Reddit Slideshow

An image browser app written in Python and Flask. Browse subreddits as image galleries and slideshows in a clean interface that allows you to focus on the picture, not on the surrounding noise.

This webapp parses the Reddit JSON API to find the top image links per week of a given subreddit. It then randomly chooses one on each page load and serves it to the user.

You can try out Reddit Slideshow on any web-enabled device, including your mobile, here. To browse a custom subreddit or set a slideshow timer, just click the subreddit or timer buttons in the top left corner. You're good to go!

You can share a URL to a subreddit and timer: /r/ITookAPicture with 10-second slideshow

Or, to share a particular photo, click the Permalink button in the bottom right: "ITAP of a creek in the Smokey Mountains"
